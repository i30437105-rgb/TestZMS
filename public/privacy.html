в отношении обработки персональных данных

1. Общие положения

1.1. Настоящая Политика конфиденциальности (далее — «Политика») определяет порядок обработки и защиты персональных данных физических лиц (далее — «Пользователи»), которые предоставляют свои данные при использовании сайтов Индивидуального предпринимателя Добрусиной Марины Валерьевны (далее — «Оператор»).

1.2. Настоящая Политика разработана в соответствии с Федеральным законом от 27.07.2006 № 152-ФЗ «О персональных данных» и иными нормативными правовыми актами Российской Федерации в области персональных данных.

1.3. Заполняя формы на сайтах Оператора и предоставляя свои персональные данные, Пользователь выражает согласие с условиями настоящей Политики.

1.4. В случае несогласия с условиями Политики Пользователь должен прекратить использование сайтов Оператора.

1.5. Настоящая Политика применяется только к сайтам Оператора. Оператор не контролирует и не несёт ответственности за сайты третьих лиц, на которые Пользователь может перейти по ссылкам, размещённым на сайтах Оператора.

2. Персональные данные, которые обрабатывает Оператор

2.1. Оператор собирает и обрабатывает следующие персональные данные Пользователей:

  • имя (фамилия, имя);
  • номер телефона;
  • данные, автоматически передаваемые при посещении сайта: IP-адрес, данные файлов cookie, информация о браузере, время доступа, адреса запрашиваемых страниц.

2.2. Оператор не осуществляет сбор и обработку специальных категорий персональных данных (сведения о расовой, национальной принадлежности, политических взглядах, религиозных убеждениях, состоянии здоровья, интимной жизни).

3. Цели обработки персональных данных

3.1. Оператор обрабатывает персональные данные Пользователей в следующих целях:

  • идентификация Пользователя для оказания информационно-консультационных услуг;
  • предоставление доступа к онлайн-практикумам, курсам, маркетинговым аудитам и иным услугам Оператора;
  • связь с Пользователем для подтверждения заказа, уточнения деталей, информирования об изменениях;
  • направление информационных и рекламных сообщений (при наличии согласия Пользователя);
  • улучшение качества услуг и удобства использования сайтов;
  • проведение статистических и аналитических исследований;
  • исполнение обязательств по договору публичной оферты.

4. Правовые основания обработки

4.1. Оператор обрабатывает персональные данные на следующих правовых основаниях:

  • согласие Пользователя на обработку персональных данных (ст. 6 ч. 1 п. 1 ФЗ-152);
  • исполнение договора, стороной которого является Пользователь (ст. 6 ч. 1 п. 5 ФЗ-152);
  • законные интересы Оператора, не нарушающие права и свободы Пользователя.

5. Порядок обработки персональных данных

5.1. Обработка персональных данных осуществляется с использованием средств автоматизации и без использования таких средств.

5.2. Оператор обеспечивает хранение персональных данных на территории Российской Федерации в соответствии с требованиями законодательства.

5.3. Персональные данные хранятся в течение срока, необходимого для достижения целей обработки, но не менее срока, установленного законодательством Российской Федерации.

5.4. По достижении целей обработки или по истечении срока хранения персональные данные уничтожаются.

6.1. Оператор может передавать персональные данные третьим лицам в следующих случаях:

  • передача необходима для защиты прав и законных интересов Оператора.

6.2. При передаче персональных данных Оператор требует от третьих лиц соблюдения конфиденциальности персональных данных.

7.1. Сайты Оператора используют файлы cookie для обеспечения корректной работы сайтов, улучшения пользовательского опыта и сбора аналитических данных.
